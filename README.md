# 🎧 AI Audiobook Generator

This project converts text into an audiobook using AI.

## 🚀 Features
- Extract text from files
- Generate audio using AI
- Save output as MP3

## 🛠️ Tech Used
- Python
- Streamlit
- Google Generative AI
- Text-to-Speech (TTS)

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python main.py
